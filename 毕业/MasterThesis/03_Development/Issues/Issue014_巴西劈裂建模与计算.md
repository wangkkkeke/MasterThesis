## 任务描述

建立巴西劈裂圆盘 FEM-DEM 耦合模型：
- 圆盘中心区域为 DEM（预期断裂区）
- 外围加载区为 FEM（需精确施力）
- InterfaceBond 连接两区域
- 验证劈裂荷载和裂纹路径

## 几何参数

```
圆盘：直径 D = 0.05m（参考2025赵旭剑论文），厚度 t = 0.005m
加载：上下两点（线）接触，速度边界 v = 0.01m/s（缓慢加载）
分区：
  FEM区：环形外圈，宽度约 0.01m
  DEM区：中心圆，半径约 0.015m
```

## 理论劈裂荷载（来自 FormulaIndex.md）

```
P_split = π × ft × D × t / 2

其中：
ft = 抗拉强度（需标定）
D = 直径
t = 厚度
```

## Gmsh 建模要点

```
1. 建圆盘（Circle），中心 (0,0,0)，半径 D/2
2. 内圆（DEM区）半径 0.015m
3. 环形区（FEM区）= 外圆 - 内圆
4. Physical Volume 分别命名 "dem_zone"、"fem_zone"
5. 加载线（上/下各一小段弧）→ Physical Surface "load_top" / "load_bot"
6. 单元尺寸：FEM区0.005m，DEM区（颗粒生成）

注意：圆弧必须 CCW（来自 DevelopmentHistory.md 踩坑3）
```

## 材料参数（参考赵旭剑2025论文）

```
E = 7.7 GPa, ν = 0.22, ρ = 2700 kg/m³
c = 20 MPa, φ = 30°, ft = 5 MPa
gi = 10 J/m², gii = 50 J/m²
```

## 测试方法

```
通过条件：
1. 裂纹路径：沿直径方向（竖直）扩展，无偏转
2. 劈裂荷载误差 < 15%（对比理论解或文献值）
3. 应力集中区在加载点和裂纹尖端（ParaView可视化确认）

工况（来自 ThesisOutline.md §5.2）：
- 工况1：弹性分析（验证应力场）
- 工况2：完整断裂（验证裂纹路径和劈裂荷载）
```
