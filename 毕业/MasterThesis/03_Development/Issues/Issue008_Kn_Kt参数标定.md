## 任务描述

确定 InterfaceBond 刚度参数 Kn 和 Kt 的取值依据与标定方法，
使界面无物理载荷时不产生明显 Ghost Force，有载荷时准确传递。

## 理论依据（来自 TheoryGuide.md）

**刚度下界（界面协调条件）：**
```
Kn ≥ α × E_FEM / h_FEM

其中：
E_FEM = FEM侧弹性模量（Pa）
h_FEM = 界面处FEM单元特征长度（m）
α     = 协调系数（建议初始值 α=1~10）
```

**时间步约束（上界）：**
```
Kn ≤ m_min / dt²

其中：
m_min = 最小颗粒质量（kg）
dt    = 时间步长（s）
```

**Kt / Kn 比值：**
```
Kt / Kn ≈ (1-2ν) / (2(1-ν))     （泊松比 ν 确定）

砂岩（ν≈0.25）：Kt/Kn ≈ 0.4
```

## 标定流程

```
Step1：计算 h_FEM（Gmsh中单元最小边长）
Step2：计算刚度下界 Kn_min = α × E / h_FEM
Step3：计算时间步约束上界 Kn_max = m_min / dt²
Step4：取 Kn = Kn_min × 2（保留余量）
Step5：取 Kt = Kn × (1-2ν)/(2(1-ν))
Step6：运行无载荷算例，Ghost Force < 1% × E × 1e-6
Step7：运行单轴拉伸，应力场与纯FEM结果误差 < 5%，确认参数有效
```

```
通过条件：
1. Ghost Force < 1% × 施加荷载（无外载荷算例）

3. 稳定性：运行1000步无发散
```
